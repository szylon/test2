Szablon dla strony Internetowej zawierający w sobie sktrukturę:

- skrypty JS,
- style CSS, SCSS
- pluginy dla podstron,

Projekt zawiera następujące prefixy:



### Skrócona instrukcja komend

Instalacja projektu:

```sh
$ git clone https://scieżka_do_wybranego_projeku <= instalacja przy użyciu połączenia po https

- lub -

$ git clone git@github.com:scieżka_do_wybranego_projeku <= instalacja przy użyciu klucza prywatnego zainstalownego na danym komputerze
```

Instalacja paczek:

```sh
$ npm i <= instalacja paczek dla wybranego projektu
$ npm i nazwa_paczki <= instalacja wybranej paczki nazwa_paczki
```

Uruchomienie projektu i kompilacja:

```sh
$ npm run watch <= praca w trybie rzeczywistym
$ npm run build <= wygenerowanie całego finalnego projektu
```

Aktualizacja:

```sh
$ ncu <= sprawdzenie lokalnych paczek czy są nowe aktualizacje
$ ncu -g <= sprawdzenie globalnych paczek czy są nowe aktualizacje

$ ncu -u <= aktualizacja paczek
```
